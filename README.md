### Como executar?

#### Executando bash script
```shell script
$ bash ./execute.sh <caminho_do_arquivo>
``` 

Exemplo
```shell script
$ bash ./execute.sh ./tests/in1.txt
``` 

#### Executando script Python
```shell script
python main.py <caminho_do_arquivo>
``` 

Exemplo
```shell script
python main.py ./tests/in1.txt
```
